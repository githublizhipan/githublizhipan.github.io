迭代器

通常来讲，从容器c获得一对迭代器的方法就是std::begin(c)和std::end(c)。我们还有std::cbegin、std::rbegin和std::crbegin，以及对应的end函数，来分别得到const或非const的、正序或反序迭代器。

顺序容器

stack

stack主要函数有：

  • top：返回栈顶
  • push：入栈
  • pop：出栈
  • empty：看看是不是空的
  • size：容器里到底有多少东西

queue

queue主要的函数有：

  • front：返回队头
  • back：返回队尾
  • push：入列
  • pop：出列
  • empty：看看是不是空的
  • size：容器里到底有多少东西

list

list主要的函数有：

  • push_front、push_back：往链表头或者链表尾插入数据

  • pop_front、pop_back：从链表头或者链表尾里拿走数据

  • size：容器里到底有多少东西

关联容器

map

map主要的函数有：

  • insert：插入一个pair，如果key已经存在的话那么返回值的第二项是false

  • find：查找一个key

  • erase：删除一个key

  • clear：清空map

  • count：返回key的数量

set

set主要的函数有：

  • insert：插入一个key，如果key已经存在的话那么返回值的第二项是false

  • find：查找一个key

  • erase：删除一个key

  • clear：清空map

  • count：返回key的数量

priority_queue

priority_queue主要的函数有：

  • push：插入一个对象并排序
  • pop：弹出对首对象。如果不指定比较方式的话，priority_queue会默认使用小于号来比较，那么pop总是弹出最大的那个。
  • top：返回队头
  • size：返回对象的数量
  • empty：返回是否为空
